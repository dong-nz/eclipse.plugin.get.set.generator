Get-Set-Generator

Eclipse plugin to generate all get and set methods for POJO in java project.

Download .jar Download source code View on GitHub

How to install

  1. Download the latest jar file.
  2. Copy jar file to plugins folder of Eclipse.
  3. Restart Eclipse.

Step 2
